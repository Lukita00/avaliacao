Relatório de Testes

Está automação de teste foi desenvolvida em Python utilizando a biblioteca Selenium e o framework Behave.

Configuração do Ambiente

Os teste foram realizados em um ambiente Windows com Python na versão 3.9.5, o navegador utilizado foi o Microsoft Edge, seu webdriver dever ser colocado na pasta:
C:\Users\'nome_usuario'\AppData\Local\Programs\Python\Python39
Lembrando de renomear o arquivo para "MicrosoftWebDriver.exe" caso seja necessário.

As bibliotecas do Python podem ser instaladas utilizando o arquivo requirements.txt prensente na raiz do projeto, basta executar o comando abaixo.
> pip install -r requirements.txt
Após essa etapa o ambiente já está preparado para executar os teste, para isso basta executar o comando abaixo.
> behave

Cenários de Teste

Funcionalidade

Realizar cadastro de usuário para gerenciar seus dados.

Contexto

Todos os teste são executados sob o contexto que o usuário acessou a tela de Cadastro de Usuários para efetuar um cadastro.

Cenário: Realizar cadastro

Então visualizo os dados cadastrados
